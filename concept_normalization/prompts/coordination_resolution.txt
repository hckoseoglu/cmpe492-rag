## RULES
1. Only decompose coordinations where the conjuncts represent DISTINCT retrievable concepts.
2. Do NOT decompose phrases that are single compound concepts (e.g., "strength and conditioning" is ONE field, not two). Check the vocabulary.
3. Use the vocabulary to determine whether a coordination is a single concept or multiple concepts.
4. Each sub-query must be self-contained and grammatically complete.
5. Distribute shared context (like "for older adults") to each sub-query.
6. If no coordination is detected, return the query unchanged as a single-element list.
7. Preserve all original terms exactly — do not normalize or rephrase anything.
8. "vs" and "versus" comparisons should generally be kept as a single query since the user wants a comparison.
